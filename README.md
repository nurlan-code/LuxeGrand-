# 🏨 LuxeGrand Hotels - Premium Hotel Network

A luxurious hotel website showcasing premium hospitality services with modern design and interactive features.

## 📋 Project Overview

LuxeGrand Hotels is a comprehensive luxury hotel website featuring multiple pages with rich content, interactive elements, and responsive design. The project demonstrates modern web development practices with embedded CSS and JavaScript for optimal performance.

## 🌟 Features

### ✨ Interactive Elements
- **Smooth Animations**: CSS animations with intersection observer
- **FAQ Accordion**: Interactive question-answer sections
- **Statistics Counter**: Animated number counting on scroll
- **Form Validation**: Real-time form validation with feedback
- **Responsive Navigation**: Mobile-friendly hamburger menu

### 🎨 Design Highlights
- **Luxury Aesthetic**: Gold (#d4af37) and elegant color scheme
- **Modern Typography**: Playfair Display and Inter font families
- **Gradient Backgrounds**: Beautiful gradient combinations
- **Card-based Layout**: Clean, organized content presentation
- **Hover Effects**: Smooth transitions and interactive feedback

### 📱 Responsive Design
- **Mobile First**: Optimized for all device sizes
- **Flexible Grid**: CSS Grid and Flexbox layouts
- **Touch Friendly**: Large buttons and touch targets
- **Cross Browser**: Compatible with modern browsers

## 🔧 Installation & Setup

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Local web server (optional, for development)

### Quick Start
1. **Clone or Download** the project files
2. **Open** `index.html` in your web browser
3. **Navigate** through the different pages
4. **Test** interactive features and responsiveness

### Development Setup
```bash
# Navigate to project directory
cd 30-website/3

# Serve files locally (optional)
python -m http.server 8000
# or
npx serve .

# Open in browser
open http://localhost:8000
```
